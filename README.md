# Skybridge_Aircargo-Project
Designed and built an end-to-end air cargo analytics solution simulating real-world freight operations across 10 airlines, 40 airports, and 120 global customers.
Key Highlights:

Designed a normalized relational database with 8 tables and 26-column fact table covering full shipment financials
Wrote 22 advanced SQL queries including CTEs, Window Functions (RANK, LAG, Running Total), and 3 reusable Views
Built a 6-page interactive Power BI dashboard with 9 DAX measures covering $39.24M revenue, 24.36% profit margin, and 64.3% on-time delivery rate
Analyzed 1,000 shipments across 6 global regions identifying top revenue routes, delay patterns, and airline performance
Produced  Business Analyst reports with strategic recommendations
